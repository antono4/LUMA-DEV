## 🔧 Custom API Key (Optional)

Untuk menggunakan API key sendiri:

1. Buka `index.html`
2. Cari baris:
```javascript
const OPENHANDS_API_KEY = 'sk-oh-xxxxx';
```
3. Ganti dengan API key Anda dari [app.all-hands.dev](https://app.all-hands.dev/settings/api-keys)
