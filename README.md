## 🔧 For Full AI Power

Untuk AI responses yang lebih powerful, configure API keys:

1. Buat file `.env`:
```bash
OPENAI_API_KEY=sk-your-key-here
ANTHROPIC_API_KEY=your-anthropic-key
```

2. Install dependencies:
```bash
pip install -r requirements.txt
```

3. Jalankan server:
```bash
python app.py
```
